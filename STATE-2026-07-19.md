## What this is
Glove80 keyboard running a custom RMK (Rust) firmware stack, replacing ZMK. Components:

- `rmk/glove80/` — firmware wrapper crate (central=LH, peripheral=RH). `./build.sh` → two uf2s.
  Still builds against **vendored subtree** `rmk/vendor/rmk` (swap pending, see below).
- `tools/glove80-control/` — host CLI (USB HID vendor protocol v1.4): version, bootloader
  (both halves, hands-free via split message), keymap, lighting, config push. 88 tests.
- Lightbench — browser IDE for lighting/keymap (launch config `.claude/launch.json`);
  TS codecs mirror the Rust host protocol.
- Conditional lighting: gates + firmware-state conditions (VBUS etc.), compositor, records.
- Keyboard config: unified TOML pushed via CLI; keymap batching for hardware limits.
